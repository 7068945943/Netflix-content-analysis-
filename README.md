# 🎬 Netflix Content Analysis

## 📌 Project Type
EDA + Unsupervised Learning (DBSCAN Clustering)

## 🎯 Objective
To analyze Netflix movies and TV shows dataset and identify patterns, trends, and clusters.

## 🔍 Key Steps
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Visualization
- Clustering using DBSCAN

## 📊 Results
- Best Parameters: eps = 0.5, min_samples = 5  
- Silhouette Score: 0.834  
- Meaningful clusters identified

## 🛠️ Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

## ▶️ How to Run
- Open the notebook in Jupyter/Colab
- Run all cells
